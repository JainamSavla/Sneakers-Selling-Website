# Nike Store
Link For Website:https://jainamsavla.github.io/Sneakers-Selling-Website/
A modern, responsive single-page website for a fictional Nike sneaker store. Browse the latest sneaker collections, view product details, and simulate a purchase with a stylish checkout form.

## Features

- **Product Slider:** Browse through the latest sneaker models with images, prices, and quick buy buttons.
- **Product Details:** View detailed information, select colors and sizes, and initiate a purchase.
- **Checkout Modal:** Enter personal and payment information in a sleek, interactive form.
- **Responsive Design:** Optimized for desktop and mobile devices.
- **Gallery & New Season:** Inspirational images and promotional banners.
- **Footer:** Useful links, newsletter subscription, and social media icons.

## Getting Started

1. **Clone or Download the Repository**
2. **Open `index.html` in your browser**

No build steps or dependencies required.

## File Structure

- `index.html` – Main website structure
- `style.css` – Styles and layout
- `app.js` – Interactive functionality
- `img/` – Images and icons

## Customization

- Replace images in the `img/` folder to update product visuals.
- Edit product details and prices directly in `index.html`.
- Update styles in `style.css` as needed.
